angular2-tdd-boilerplate
===================

Boilerplate for Angular2 with TDD (Jasmine & Karma)

Prepare
------
Install dependencies via npm
```````
$ npm install
```````

Run
------
To run a web server and automatic transpiling
``````
$ npm start
``````
To run transpiling
``````
$ npm run tsc
``````
To run a web server
``````
$ npm run lite
``````

Tests
-----
To run the tests with Karma
``````
$ npm test
``````
